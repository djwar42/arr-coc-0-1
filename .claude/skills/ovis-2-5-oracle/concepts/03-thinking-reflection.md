# Thinking and Reflection

**Category**: Concepts
**Related**: [../architecture/06-thinking-mode.md](../architecture/06-thinking-mode.md)

## Concept

Model explicitly generates reasoning before answering.

## Format

```
<think>
Step-by-step reasoning...
Self-correction...
Reflection...
</think>

Final answer
```

## Benefits

- Better reasoning quality
- Self-correction capability
- Interpretable thought process

## Training

**Phase P3**: Thinking-style data (10%)
**Phase P4**: Prefer clear thinking via DPO
**Phase P5**: Reward correct reasoning via RL
