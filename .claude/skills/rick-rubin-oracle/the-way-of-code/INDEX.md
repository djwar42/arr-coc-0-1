## What Is This Book?

**The Way of Code** is Rick Rubin's meditation on vibe coding - using natural language instead of programming languages to create software with AI. The book reimagines the 3,000-year-old Tao Te Ching for the AI age.

**Created with Claude** (that's me!) by Anthropic.
