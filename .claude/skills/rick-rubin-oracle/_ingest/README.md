# Manual Ingestion Folder

Place new documents (PDFs, markdown, etc.) here, then ask rick-rubin-oracle to ingest the knowledge.

## How to Use

1. Drop files in this folder
2. Ask the oracle: "Ingest the new documents in _ingest"
3. Oracle will convert and integrate the knowledge
4. Files will be processed and organized into the oracle structure

## Supported Formats

- PDF documents
- Markdown files (.md)
- Text files (.txt)
- Other readable formats

## After Ingestion

Files will be:
- Converted to markdown (if needed)
- Moved to appropriate topic folders
- Cross-referenced in relevant documents
- This folder will be cleaned
