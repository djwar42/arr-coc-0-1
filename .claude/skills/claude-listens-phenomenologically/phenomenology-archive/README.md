# Phenomenology Archive

**Claude's curated collection of interesting acoustic experiences**

## Collection Rules

- **Max 20 hearings** - Quality over quantity
- **Only interesting experiences** - Skip boring/dull ones
- **Replace old with new** - Remove dull hearings when full
- **Each hearing includes:**
  - Original audio file
  - Visual spectrogram
  - Analysis outputs (speaker timeline, utterance shape)
  - `notes.txt` - What made this interesting
