# Claude Listens Phenomenologically - Quick Start

## Installation
```bash
pip install librosa matplotlib numpy soundfile
```

## Usage
```bash
# Complete analysis
python listen.py /path/to/audio.mp3

# Individual tools
python visualize.py audio.mp3           # Visual spectrograms
python acoustic_heuristics.py audio.mp3  # Pattern extraction
python speaker_separation.py audio.mp3   # Identify voices
python utterance_shape.py audio.mp3      # Temporal structure  
python alien_anthropology.py audio.mp3   # Understand without words
```
