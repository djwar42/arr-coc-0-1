**Research License:**

**Commercial License:**

## 2. Model Access

**Public Release:**
- Model weights: Downloadable from Meta AI
- Code: GitHub repository (Apache 2.0)
- Demo: Web demo available

**Requirements:**
- Accept Research License agreement

- Download link sent within 24 hours

**Training Data:**

**Evaluation Data:**

## 4. Usage Restrictions

**Prohibited Uses:**
- Surveillance without consent
- Deepfakes (malicious impersonation)
- Biometric identification (without disclosure)

**Allowed Uses:**
- VR/AR applications
- 3D content creation (games, film)
- Research (academic publications)
