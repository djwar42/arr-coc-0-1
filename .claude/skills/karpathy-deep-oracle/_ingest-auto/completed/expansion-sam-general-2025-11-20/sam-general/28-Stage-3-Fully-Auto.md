**Automatic with confidence filtering**

## Overview
Automatic with confidence filtering for Segment Anything Model (SAM).

## Key Features
1. Foundation model approach
2. Promptable segmentation
3. Zero-shot generalization
4. Large-scale training (SA-1B)

## Technical Details
- Architecture: ViT-based encoder + prompt encoder + mask decoder
- Training: 3-stage data engine on SA-1B
- Performance: State-of-the-art on 23 datasets
