## Common Patterns & Widget Usage

### Widget Patterns Identified

**Data Display**:
- `DataTable` - Used in Harlequin (SQL results), Dolphie (MySQL metrics), RTUI (ROS topics)
- `RichLog` - Common in log viewers (Logmerger, PDFalyzer)
- `Tree` - File browsers (Browsr), AST viewers (Textual-Astview)

**Input & Forms**:
- `Input` with validation - Harlequin (SQL queries), Words-TUI (writing)
- `Select` dropdowns - Terraform-TUI, Django-TUI
- `Autocomplete` - Textual-Autocomplete library

**Progress & Monitoring**:
- `ProgressBar` - OF-Scraper, XHS-Downloader (download progress)
- Live metrics - Memray (memory), Dolphie (database stats)
- Real-time updates - Traffic (air traffic), Bluesky (simulation)

**Specialized**:
- Canvas rendering - Textual-Paint (pixel art), Netext (graphs)
- Markdown display - Frogmouth (browser), Markata (preview)
- Terminal emulation - Textual-Terminal (embedded shells)
