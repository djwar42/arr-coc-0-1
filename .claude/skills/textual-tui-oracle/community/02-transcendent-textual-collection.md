# Transcendent Textual Collection

Comprehensive collection of Textual applications, libraries, tools, and resources curated by Dave Pearson (davep). This collection served as a major community resource from its creation until archival on June 22, 2024.

**Source**: [GitHub - davep/transcendent-textual](https://github.com/davep/transcendent-textual)
**Stars**: 185 | **Forks**: 37
**Status**: Archived (read-only as of June 22, 2024)
**Accessed**: 2025-11-02

---

## Overview

The Transcendent Textual collection aimed to gather links to as many Textual-based applications, third-party libraries, and tools as possible. It provides a comprehensive snapshot of the Textual ecosystem, organized by resource type.

**Collection Categories**:
- Official Textualize libraries and tools
- Third-party widget libraries and extensions
- Production applications built with Textual
- Learning resources and tutorials

## Official Textualize Libraries & Tools

Official tools and libraries maintained by Textualize for Textual development.

### Development & Testing Tools

**[pytest-textual-snapshot](https://github.com/Textualize/pytest-textual-snapshot)**
- Snapshot testing framework for Textual applications
- Enables visual regression testing
- Captures and compares UI states

**[textual-dev](https://github.com/Textualize/textual-dev)**
- Official development tools for Textual
- CLI utilities for Textual app development
- Debugging and inspection capabilities

**[Textual Syntax Highlighter for VSCode](https://marketplace.visualstudio.com/items?itemName=Textualize.textual-syntax-highlighter)**

- Official marketplace extension
