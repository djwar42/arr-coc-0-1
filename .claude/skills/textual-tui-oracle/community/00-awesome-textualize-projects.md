# Awesome Textualize Projects - Curated Collection

## Overview

This document catalogs curated Textual TUI applications from the community-maintained awesome-textualize-projects list. Projects are organized by category with GitHub stars, descriptions, and notable features.

**Source**: [awesome-textualize-projects](https://github.com/oleksis/awesome-textualize-projects) (93 stars, accessed 2025-11-02)

---

## Official Textualize Projects

### Textual Framework
- **Repository**: [Textualize/textual](https://github.com/Textualize/textual)
- **Description**: TUI (Text User Interface) framework for Python inspired by modern web development
- **Documentation**: [textual.textualize.io](https://textual.textualize.io/)
- **Official Gallery**: [textualize.io/textual/gallery](https://www.textualize.io/textual/gallery)

### Rich Library
- **Repository**: [Textualize/rich](https://github.com/Textualize/rich)
- **Description**: Python library for rich text and beautiful formatting in the terminal
- **Documentation**: [rich.readthedocs.io](https://rich.readthedocs.io/en/latest/)
- **Features**: Tables, panels, progress bars, trees, syntax highlighting, pretty printing
- **Official Gallery**: [textualize.io/rich/gallery](https://www.textualize.io/rich/gallery)

### Rich CLI
- **Repository**: [Textualize/rich-cli](https://github.com/Textualize/rich-cli)
- **Description**: Command line toolbox for fancy terminal output
- **Features**: Syntax highlighting for many file types, special support for markdown, JSON, and CSV tables

## Notable Patterns & Trends

### Popular Use Cases:
1. **Developer Tools** - Strong focus on code inspection, database management, and developer workflows
2. **Content Viewing** - Markdown browsers, image viewers, and log file analysis tools
3. **Widget Libraries** - Extensive ecosystem of reusable UI components
4. **Games** - Surprising number of entertainment applications (solitaire, piano, paint)

### Common Widget Usage:
- **DataTable**: Log viewers, database tools
- **Input & Forms**: Note-taking apps, configuration tools
- **Custom Widgets**: Canvas, terminal emulators, autocomplete
- **Tree Views**: File browsers, JSON visualizers

### Integration Patterns:
- **External APIs**: ChatGPT (Elia), Pinboard (Tinboard), Django (Django-TUI)
- **Hardware**: Bluetooth LE (HumBLE Explorer)
- **File Systems**: File browsers (Kupo, Fspicker), recovery tools (RecoverPy)
- **Version Control**: Git integration (Dunk)

### Standout Projects:
- **Harlequin** - Full SQL IDE in terminal (complex data interaction)
- **RecoverPy** - System-level file recovery (advanced filesystem operations)
- **Elia** - AI integration (modern API usage)
- **Toolong** - Official log viewer (production-grade tool)
- **Textual-Paint** - Complex graphics manipulation (MS Paint clone)
