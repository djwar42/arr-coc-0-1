**Accessed**: 2025-11-02

## What is Textual?

Textual is a **Rapid Application Development (RAD)** framework for Python, built by [Textualize.io](https://www.textualize.io).

Build sophisticated user interfaces with a simple Python API. Run your apps in the **terminal** or a **web browser** (via [textual-serve](https://github.com/Textualize/textual-serve)).

## Key Features

### **Rapid Development**
Uses your existing Python skills to build beautiful user interfaces.

### **Low Requirements**
Run Textual on a single board computer if you want to.

### **Cross Platform**
Textual runs just about everywhere.

### **Remote**
Textual apps can run over SSH.

### **CLI Integration**
Textual apps can be launched and run from the command prompt.

### **Open Source**
Textual is licensed under MIT.

## Built with Textual - Showcase Applications

The Textual ecosystem includes many production-ready applications and tools:

### 1. Posting
**The API client that lives in your terminal.**

A beautiful open-source terminal app for developing and testing APIs.

- **Website**: [https://posting.sh/](https://posting.sh/)
- **GitHub**: [https://github.com/darrenburns/posting](https://github.com/darrenburns/posting)

**Features**:
- HTTP methods (GET, POST, PUT, DELETE, etc.)
- Request headers, body, query parameters, authentication
- Response viewing (body, headers, cookies, scripts, trace)
- Collection management
- Visual request/response handling

### 2. Toolong
**A terminal application to view, tail, merge, and search log files (plus JSONL).**

- **GitHub**: [https://github.com/textualize/toolong](https://github.com/textualize/toolong)

**Features**:
- View and tail log files
- Merge multiple logs
- Search functionality
- JSONL support
- Real-time feed monitoring
- Case-sensitive and regex search

### 3. Memray
**Memory profiler for Python, built by Bloomberg.**

- **GitHub**: [https://github.com/bloomberg/memray](https://github.com/bloomberg/memray)

**Features**:
- Live tracking of memory usage
- Heap usage visualization
- Thread analysis
- Allocation tracking
- Total bytes, own bytes, allocation counts
- Location-based memory profiling

### 4. Dolphie
**Your single pane of glass for real-time analytics into MySQL/MariaDB & ProxySQL.**

- **GitHub**: [https://github.com/charles-001/dolphie](https://github.com/charles-001/dolphie)

**Features**:
- Real-time database monitoring
- Host information display
- System utilization (CPU, memory, disk I/O, network)
- InnoDB metrics
- Binary log monitoring
- Query statistics
- Process list viewer
- Metric graphs with historical data

### 5. Harlequin
**An easy, fast, and beautiful database client for the terminal.**

- **Website**: [https://harlequin.sh/](https://harlequin.sh/)

**Features**:
- Data catalog browser
- Query editor
- Query results display
- Multiple database support
- Schema exploration
- Interactive query execution
